# Debug Changelog — 20260403_131211_808529

**Issue:** In 3rd person mode, all guns look identical and melee weapons are invisible.

## Root Causes Found

### Bug 1 — All guns look identical
`main.js` created a single static `tpGunGroup` with a hardcoded generic gray box + barrel at setup time. It was never rebuilt when the player switched weapons. Only its `.visible` flag was toggled.

### Bug 2 — Melee weapons invisible
The visibility check `tpGunGroup.visible = !!(wDef && wDef.type !== 'melee' && curWep !== 'fists' && ...)` explicitly hid the gun group for all melee/fist weapons with no alternative mesh shown.
